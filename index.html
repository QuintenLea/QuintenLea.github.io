Wat is informatica?

Informatica is eigelijk het verwerken van informatie in een computer. Het word ook wel ICT (Informatie- en Communicatietechnologie) genoemd. Programmeren doe je vaak met een programmeertaal. Enkele bekende programmeertalen zijn Java Python en Javascript. Programmeertalen gebruik je om opdrachten te beschrijven die de computer moet uitvoeren. Deze site is bijvoorbeeld gemaakt met de programmeertaal html.
